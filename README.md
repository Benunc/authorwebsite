# Ben Meredith — Author site

Simple static site for [Ben Meredith](https://benlikes.us/ssbook), hosted on GitHub Pages.

## Contents

- **index.html** — Single-page author site (bio, book link, contact)
- **style.css** — Layout and typography (no external dependencies)
- **BenHeadshot2.jpg** — Author headshot

## Deploy on GitHub Pages

1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Under "Build and deployment", set **Source** to **Deploy from a branch**.
4. Choose branch **main** (or your default) and folder **/ (root)**.
5. Save. The site will be at `https://<username>.github.io/<repo>/` (or your custom domain if configured).

No build step required—GitHub Pages serves the files as-is.
